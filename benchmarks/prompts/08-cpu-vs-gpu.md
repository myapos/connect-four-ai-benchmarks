Prompt1

You are a senior frontend engineer.

Create a Vue 3 + TypeScript single-file component for a simple Todo List.

Requirements:

- Use `<script setup lang="ts">`
- Define a Todo type with id, title, completed
- Allow adding a new todo
- Allow toggling completed state
- Allow deleting a todo
- Show a computed count of remaining todos
- Prevent adding empty todos
- Use basic accessible markup
- Keep the code in one .vue file
- Do not use external libraries

Return only the Vue component code and a short explanation.
