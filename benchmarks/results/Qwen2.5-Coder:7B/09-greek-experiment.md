React Suspense είναι μια δυνατότητα του React για τη σύγχυση και την ανάπτυξη ισχυρών UIs που σε βοηθούν να είστε επιτυχής και αξιόπιστος στις χρήστες σας. Μέσω της Suspense, μπορούμε να δημιουργήσουμε UIs που είναι βιώσιμη και δυνατό να αναπτύξονται σε χρόνο.

Το Suspense το φέρει να αλλάξει μια απλή ιδέα: "Η UI είναι βιώσιμη και μπορεί να αναπτύξεται". Εάν το React συγκρίνει ότι το δίκτυο τους δεν έχει φέρει κάτι νέο (ή μηνυματισμό), τότε μπορεί να αλλάξει την UI για να δείξει ότι είναι σε βιώσιμη.

Το Suspense έχει δύο κρίνουμε μέσω του React.lazy() (για δικαίως οριζόμενη φόρτωση στο χρόνο) και της Suspense API:

1. React.lazy(): Δημιουργεί ένα λέξι με τη δυνατότητα να φορτώσει ένα είδος JavaScript κωδικού στο χρόνο.
2. Suspense API: Παράγει μια UI για τη διαδικασία φόρτωσης.

Όταν χρησιμοποιώ το React Suspense;

Το React Suspense είναι αξιόπιστος για πολλές λειτουργίες, με συγκεκριμένη ανάλυση:

- Φορτώσει και διχαύξει με τη δυνατότητα δικαίως σε χρόνο (React.lazy())
- Διαδίδει πληροφορίες για την ανάπτυξη UI και για να μπορήσουμε να δείξουμε ένα ρωμαϊκό UI σε χρόνο
- Διευκολύνει την δημιουργία πιο ανάπτυσσαστικών, ιδιοχωρισμένων UI

Όταν μήπως χρησιμοποιώ το React Suspense;

Ο React Suspense δεν είναι καλά προσωπικά για:

- Προκλήσεις στην απόδοση
- Συγχύσεις με το Διαδίκτυο (με διαφορετικές προσωπικές λειτουργίες)
- Προγνώμονες για οποιαδήποτε συνέχεια χωρίς δημιουργία UI (όπως το μη κάνω με την Suspense API)

Επιπλέον, αφού ο React Suspense είναι νέα γνώση, θα πρέπει να χρησιμοποιήσετε με προσοχή και να βάλετε σημεία υπόρρησης.

Παράδειγμα:

import React, { lazy, Suspense } from 'react'; const OtherComponent = lazy(() => import('./OtherComponent')); function MyComponent() { return ( <Suspense fallback={<div>Loading...</div>}> <OtherComponent /> </Suspense> ); }
